# AI Skills

A look at how I actually use AI for real work — not demos.

This repo is organised into three parts:

- **[`/projects`](./projects)** — what I've been building recently: an Airtable system for a
  multi-million dollar programme, a self-built health app, a personal dashboard, automations, and
  an econometric study turned into a podcast for practitioners.
- **[`/skills`](./skills)** — two Claude skills I built and use for real problems I had: one
  turns years of research expertise into a repeatable analysis process, the other automates a
  small but recurring piece of admin end to end without me re-explaining the rules from scratch
  in each conversation.
- **[`/agency`](./agency)** — not something I built, but how I think and act: the choices behind
  leaving an academic career, building and then letting go of a business, and what ties it
  together.

# AI Skills

A look at how I actually use AI for real work — not demos.

This repo is organised into two parts:

- **[`/projects`](./projects)** — what I've been building recently: an Airtable system for a
  multi-million dollar programme, a self-built health app, a personal dashboard, automations, and
  an econometric study turned into a podcast for practitioners.
- **[`/skills`](./skills)** — two Claude skills I built and use for real problems I had: one turns years of research expertise into a repeatable analysis  process, the other automates a small but recurring piece of admin end to end without me re-explaining the rules from scratch in each conversation.
